术语翻译
-----------------
翻译术语的时候请参考这个流程：

	1,	尽量保证和已翻译的内容一致
	2,	尽量先搜索，一般来说编程语言的大部分术语是一样的，可以参考微软官方术语搜索http://www.microsoft.com/Language/zh-cn/Search.aspx 
	3,	如果以上两条都没有找到合适的结果，请自己决定一个合适的翻译或者直接使用英文原文，后期校对的时候会进行统一

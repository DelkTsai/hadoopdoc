术语翻译
=========
<p>翻译术语的时候请参考这个流程：</p>


	<ul>
	<li>尽量保证和已翻译的内容一致<li>
	<li>尽量先搜索，一般来说编程语言的大部分术语是一样的，可以参考<a href="http://www.microsoft.com/Language/zh-cn/Search.aspx">微软官方术语搜索</a></li>
 	<ul>
	*如果以上两条都没有找到合适的结果，请自己决定一个合适的翻译或者直接使用英文原文，后期校对的时候会进行统一
